# Instagram Clone - Form and Profile  :computer:

## English  

Project made with HTML, CSS (flexbox) and vanilla JS.  
Although the main idea is to clone Instagram login and form interfaces, I added some extra features according to my personal preferences. 
This project is my version of Digital Innovation One Bootcamp "Decola Dev Avanade 2021" front-end challenge.

## Português

Projeto feito com HTML, CSS (flexbox) e vanilla JS. Embora a ideia principal aqui fosse reproduzir as páginas de login (form e perfil) do Instagram, tomei a liberdade de adicionar alguns recursos extras. Esse projeto é minha versão de um dos desafios do Bootcamp "Decola Dev Avanade 2021", organizado pela Digital Innovation One em parceria com a Avanade.
